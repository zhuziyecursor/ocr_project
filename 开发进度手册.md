## 六、运行说明

### 6.1 服务启动

```bash
# 后端（在项目根目录）
PYTHONPATH=. python3 -m uvicorn backend.main:app --host 0.0.0.0 --port 8082 --reload

# Celery Worker
PYTHONPATH=. python3 -m celery -A backend.tasks.document_tasks:celery_app worker --loglevel=info

# OpenDataLoader Hybrid Server（必须先启动）
opendataloader-pdf-hybrid --port 5002 --force-ocr --ocr-lang "zh,en"

# 前端
cd frontend && npm run dev   # 启动在 localhost:3000，代理到 :8082
```

### 6.2 依赖服务

| 服务 | 默认地址 | 说明 |
|------|---------|------|
| PostgreSQL | `localhost:5432` | 数据库，DB: ocr_services |
| Redis | `localhost:6379` | Celery broker + backend |
| OpenDataLoader Hybrid | `localhost:5002` | OCR 引擎，必须在处理前启动 |
| 前端 Dev Server | `localhost:3000` | Vite，API 代理到 :8082 |
| 后端 API | `localhost:8082` | FastAPI + uvicorn |

### 6.3 配置文件

核心配置在 `backend/core/config.py`，支持通过 `.env` 文件覆盖：

```env
DATABASE_URL=postgresql+asyncpg://ocr_user:ocr_password@localhost:5432/ocr_services
REDIS_URL=redis://localhost:6379/0
STORAGE_TYPE=local
LOCAL_STORAGE_PATH=/data/recognition_results
```
